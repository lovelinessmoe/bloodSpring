# BloodManger
血站数字化信息管理平台

使用SpringBoot作为框架

SpringSecurity+Jwt作为鉴权认证

使用redis作为验证码存储（我知道大材小用了）

使用knife4j作为api文档引擎前端

使用Mybatis+MybatisPlus作为持久层
